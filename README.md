# youziauth：西南大学校园网自动认证工具

youziauth 是一款面向西南大学校园网 ePortal 环境的 Windows 自动认证工具。完成一次配置后，它可以在电脑开机时自动检查网络并进行认证，同时提供系统托盘、失败重试和本地凭据保护。

> [!IMPORTANT]
> 本项目由个人开发，是非官方开源工具，与西南大学及校园网运营商无隶属、授权或合作关系。

## 功能特点

- 开机后自动检查并认证校园网；
- 已经能够正常访问互联网时自动跳过认证；
- 支持账号、密码和检测间隔设置；
- 支持系统托盘和后台持续检测；
- 认证失败时提供“重新认证”和“打开设置”通知；
- 安装版使用 Windows DPAPI 在本机保护密码；
- 不包含遥测或第三方数据收集功能。

## 使用要求

- Windows 10 或 Windows 11；
- 已连接西南大学校园网；
- 拥有可正常使用的校园网账号；
- 安装和启用开机自启动时可以接受 Windows 管理员授权提示。

普通用户安装 MSI 后不需要另行安装 Python。

## 下载与安装

1. 打开本仓库右侧的 **Releases** 页面；
2. 进入最新版本，下载 `youziauth.msi`；
3. 双击安装包并按照提示完成安装；
4. 从 Windows 开始菜单启动 **youziauth**。

安装过程中如果 Windows 显示安全确认，请先核对文件来源和 Release 页面提供的 SHA-256，再决定是否继续。

## 首次使用

1. 确保电脑已经连接西南大学校园网；
2. 在“账号”中填写校园网账号；
3. 在“密码”中填写校园网密码；
4. “检测间隔（秒）”建议保留默认值 `60`；
5. 点击 **保存**；
6. 点击 **检测一次**，确认状态显示“已认证或登录成功”；
7. 如果希望开机自动认证，勾选 **开机自启动**，然后再次保存。

启用或关闭开机自启动时，Windows 会请求管理员授权。这是因为程序需要创建或移除开机认证任务。

## 日常使用

关闭主窗口不会退出程序，而是把它隐藏到系统托盘。托盘图标会显示当前状态，例如在线、离线、检测中、已停止或错误。

右键单击托盘图标可以：

- 查看当前状态；
- 显示主界面；
- 进入设置；
- 立即检测；
- 退出程序。

如果重复启动 youziauth，程序会提示当前电脑已经在运行。此时请在系统托盘中打开已有实例。

## 设置说明

| 设置项 | 说明 |
|---|---|
| 账号 | 西南大学校园网账号 |
| 密码 | 校园网密码；安装版使用 Windows DPAPI 在本机保护 |
| 检测间隔（秒） | 后台认证成功后的常规检查间隔，建议使用默认值 60 |
| 开机自启动 | 开机时启动系统认证代理，登录 Windows 后启动托盘程序 |

“日志”区域用于查看最近的运行情况。点击 **刷新** 可以重新读取日志。

## 常见问题

### 点击“检测一次”后认证失败

请依次确认：

1. 当前连接的是西南大学校园网，而不是普通宽带或手机热点；
2. 校园网账号和密码可以在学校认证页面正常登录；
3. 账号没有欠费、停机、达到设备数量限制或被临时锁定；
4. 重新填写密码并保存，然后再次点击 **检测一次**；
5. 查看窗口中的状态和日志，确认服务端返回的具体提示。

如果校园网认证页面或协议发生变化，旧版本可能暂时无法使用。请查看 Releases 中是否有更新版本。

### 提示服务类型错误

从源码运行时，可以把配置文件中的 `service` 留空后重试：

```ini
service =
```

安装版没有暴露该高级选项。如果持续出现此错误，可以先使用学校网页认证，并关注后续版本。

### 开机后没有自动认证

- 打开设置，确认 **开机自启动** 已勾选并保存；
- 如果保存时出现管理员授权窗口，请选择允许；
- 在主界面点击 **检测一次**，先确认账号本身能够认证；
- 如果使用手机热点或其他已经可以上网的连接，程序会主动跳过校园网认证。

### 关闭窗口后找不到程序

主窗口关闭后会隐藏到系统托盘。点击任务栏右侧的托盘展开按钮，找到 youziauth 图标，然后右键选择 **显示主界面**。

### 收到“校园网认证失败”通知

通知中可以选择：

- **重新认证**：立即重新尝试；
- **打开设置**：检查账号、密码和日志；
- **本次不再提醒**：在本次 Windows 启动期间停止重复提醒。

下次重新启动 Windows 后，提醒抑制会自动重置。

## 安全与隐私

youziauth 不包含遥测或第三方数据收集功能。认证时，账号和密码只会发送到配置的校园网认证地址。

安装版将配置和 DPAPI 凭据保存在：

```text
%ProgramData%\youziauth\config.ini
%ProgramData%\youziauth\credential.dat
```

密码不会以明文形式写入安装版的 INI 文件。DPAPI 凭据只能在相应 Windows 环境中解密，但仍不应复制或分享给他人。

向他人寻求帮助前，请隐藏或删除以下内容：

- 校园网账号和密码；
- 完整登录地址及其查询参数；
- MAC 地址、设备 IP 和其他网络标识；
- 未经处理的配置文件、凭据文件、日志和截图。

安全问题报告说明见 [SECURITY.md](SECURITY.md)。

### 环境要求

- Windows 10 或 Windows 11；
- Python 3.10 或更高版本；
- 使用桌面界面时，Python 需要包含可用的 Tkinter/Tcl。

复制示例配置：

```powershell
Copy-Item .\config.example.ini .\config.ini
```

编辑 `config.ini`，填写账号并保持 `password` 为空。推荐只在当前 PowerShell 会话中设置密码：

```powershell
$env:CAMPUS_AUTH_PASSWORD = "your-password"
python .\campus_auth.py --config .\config.ini --once --verbose
Remove-Item Env:\CAMPUS_AUTH_PASSWORD
```

也可以使用隐藏输入的一次性认证脚本：

```powershell
powershell -ExecutionPolicy Bypass -File .\run_once_prompt_password.ps1
```

启动桌面设置界面：

```powershell
python .\campus_auth_gui.py
```

如果当前 Python 缺少可用的 Tkinter/Tcl，可以使用只监听本机地址的浏览器设置界面：

```powershell
python .\campus_auth_web.py
```

然后打开终端显示的本地地址，通常为 `http://127.0.0.1:8765/`。
